# miv
"Miv, its a movable, magical, div!"

## Overview
A picture based menu system, for ease of access.

## Installation
- cd 'vagrant/'
- vagrant up miv

## Shortcuts
- [esc] escape
- [e]   edit
- [i]   input
- [l]   lock
- [n]   navigation
- [r]   refresh
- [s]   settings
- [t]   url titles
